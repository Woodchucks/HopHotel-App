# HopHotel-App

----   Hotel Booking Application Exercise   ----

Front - end: Angular 11 /n
Back - end: .NET Core /n
Database: SQL Server

Main Features:

Admin:
- Adding/ Editing/ Deleting hotel
- Adding/ Editing/ Deleting room
- Adding/ Editing/ Deleting user

User
- Searching hotels via city, price, date
- Booking a hotel (+ choosing a room)
- Viewing all booking Details
- Editing/ Canceling booking
